Detailed Description

An animation. Animations are composed of a source image and an array of frames, which are just rectangle positions on the source image.

Constructor & Destructor Documentation

Animation::Animation ( )

Construct a new Animation object, without any frames or spritesheet. The frame array is not even created, which means that this object should not be used at all until frames are created with Animation::initFrames()

◆ Animation() [2/3]

Animation::Animation ( SDL_Texture *  spritesheet_)

Construct a new Animation object, without any frames. The frame array is not even created, which means that this object should not be used at all until frames are created with Animation::initFrames(). Delegates to Animation::Animation().
